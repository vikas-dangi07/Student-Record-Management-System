# 🎓 Student Record Management System

## 📌 Project Overview
The **Student Record Management System** is a web-based application designed to manage student information efficiently. It allows users to add, update, delete, and view student records through a simple and user-friendly interface. This system helps streamline academic data handling for educational purposes and mini-project submissions.

---

## 🚀 Features
- Add new student records
- View all stored student details
- Update existing student information
- Delete student records
- Simple and responsive interface
- Fast processing using Flask backend

---

## 🛠 Technologies Used
- Python (Flask)
- HTML
- CSS
- JavaScript

## ▶️ How to Run the Project Locally

### Step 1: Clone the Repository

```
git clone https://github.com/vikas-dangi07/Student-Record-Management-System.git
```

### Step 2: Open Project Folder

```
cd Student-Record-Management-System
```

### Step 3: Install Dependencies

```
pip install -r requirements.txt
```

### Step 4: Run Application

```
python app.py
```

### Step 5: Open in Browser

```
http://127.0.0.1:5000
```
